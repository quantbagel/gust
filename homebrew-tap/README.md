# Homebrew Tap for Gust

This is the official Homebrew tap for [Gust](https://github.com/quantbagel/gust), a blazing fast Swift package manager written in Rust.

## Installation

```bash
brew tap quantbagel/gust
brew install gust
```

## Upgrading

```bash
brew update
brew upgrade gust
```

## Uninstalling

```bash
brew uninstall gust
brew untap quantbagel/gust
```

## What Gets Installed

- `gust` binary
- Shell completions (bash, zsh, fish)
- Man page (`man gust`)

## Building from Source

If you prefer to build from source:

```bash
brew install --build-from-source gust
```
